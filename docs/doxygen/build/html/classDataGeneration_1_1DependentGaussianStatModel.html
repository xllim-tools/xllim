Detailed Description

A class representing a statistical model using the generated X matrix to add noise to Y.

This class generates X matrix using a data generator. This matrix is used to calculate Y through the functional model to which an X dependent noise is added. The class requires the type of the data generator to use, the noise effect in percentage and a seed used by generators.
