Detailed Description

template<typename TGamma, typename TSigma>
class EmEstimator< TGamma, TSigma >

Expectation-Maximization based estimator.

This estimator uses the algorithm Expectation-Maximization to train the GLLiM model. The E-step is referred by the method 'next_rnk' and the M-step by the method next_theta.

High-Dimensional Regression with Gaussian Mixtures and Partially-Latent Response Variables. Statistics and Computing 25(5): 893-911, September 2015.
