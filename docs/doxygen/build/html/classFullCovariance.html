Detailed Description

This class stands for a covariance matrix with no constraint.

The class stores the symmetric positive definite covariance matrix as a mat object. In order to faster the execution time it overrides all the basic operations on matrices like addition and multiplication, the computation of the determinant and the inverse of the matrix.
