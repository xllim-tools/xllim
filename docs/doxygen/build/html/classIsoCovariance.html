Detailed Description

This class stands for a covariance matrix where all the covariances are put to zero and all the variances are equal.

In order to faster the execution time and reduce the memory size needed to store this type of covariance matrix, this class stores only the variance from the matrix as a scalar. It overrides all the basic operations on matrices like addition and multiplication, the computation of the determinant and the inverse of the matrix.
