GLLiM

This page describes the GLLiM methods implying setting GLLiMParameters.

Setters

setParams(theta)

Set the parameters of the GLLiM model.

Parameters:

theta (GLLiMParameters) – The model parameters to set.

setParamPi(Pi)

Set the mixture coefficients Pi.

Parameters:

Pi (ndarray) – The mixture coefficients to set.

setParamA(A)

Set the parameter matrix A.

Parameters:

A (ndarray) – The parameter matrix to set.

setParamB(B)

Set the parameter matrix B.

Parameters:

B (ndarray) – The parameter matrix to set.

setParamC(C)

Set the parameter matrix C.

Parameters:

C (ndarray) – The parameter matrix to set.

setParamGamma(Gamma)

Set the gamma parameters.

Parameters:

Gamma (ndarray) – The gamma parameters to set. Shape depends on Gamma constraints.

setParamSigma(Sigma)

Set the sigma parameters.

Parameters:

Sigma (ndarray) – The sigma parameters to set. Shape depends on Sigma constraints.
